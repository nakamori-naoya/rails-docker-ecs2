【プロフィール】

`プロフィールの作成`  
```
システムは、「ログイン・新規登録」ページを表示する。
顧客は「ログイン・新規登録」ページで、顧客情報(メールアドレスとパスワード)を入力し、「新規登録」ボタンをクリックする。
システムは、メールアドレスのフォーマットやメールアドレスが一意なものか検証し、パスワードと確認用パスワードが一致するか検証し、「顧客プロフィール作成」ページを表示する。

顧客は、プロフィール情報(ニックネーム・画像・説明文・プログラミングを開始した年月・ウェブサイトのURL)を入力し、「登録」ボタンをクリックする。
システムは、ニックネームの長さが長すぎないか、重複するニックネームはないか、プログラミング開始年月は適正な数値か、画像の大きさが適切かどうかを検証する。
システムは、「顧客プロフィール詳細」ページを表示する。

【代替コース】
<メールアドレスのフォーマットが正しくない場合>
システムは、「送信」ボタンを押せないようにする。

<メールアドレスがすでに登録されている場合>
システムは「メールアドレスはすでに登録されています。」メッセージを返す。

<パスワードと確認用パスワードが一致しない場合>
システムは、「送信」ボタンを押せないようにし、「パスワードが一致しません」メッセージを返す。

<ニックネームの入力、プロフィール画像の選択ともにない場合>
システムは、「送信」ボタンを押せないようにする。

<ニックネームの長さが長すぎる場合>
システムは、「送信」ボタンを押せないようにする。

<重複するニックネームがある場合>
システムは「同じニックネームのユーザーが存在しました。ニックネームを変更して下さい」メッセージを返す。

<プログラミング開始年月は適正な数値ではない場合>
システムは、「不正な値が記入されています。正しい数値を選択して下さい」メッセージを返す

<画像の大きさが適切でない場合>  
システムは「画像の大きさが適切ではありません。画像を選択し直して下さい」メッセージを返す。
```


`プロフィールの編集`
```
【基本コース】
顧客は、「ナビゲーションバー」から「アイコンボタン」をクリックする。
システムは、顧客のプロフィール情報を取得し、「顧客プロフィール詳細」ページを表示する。
顧客は、「編集」ボタンをクリックする。
システムは、「顧客プロフィール編集」ページを表示する。
顧客は、ニックネームの入力またはプロフィール画像の選択を行い、プロフィールの説明文を入力し、「送信」ボタンをクリックする。 ※プロフィールの説明文は必須ではない。
システムは、ニックネームの長さが長すぎないか、重複するニックネームはないか、プログラミング開始年月は適正な数値か、画像の大きさが適切かどうかを検証する。
システムは、自身の「顧客プロフィール詳細」ページを表示する。

【代替コース】
<ニックネームの入力、プロフィール画像の選択ともにない場合>
システムは、「送信」ボタンを押せないようにする。

<ニックネームの長さが長すぎる場合>
システムは、「送信」ボタンを押せないようにする。

<重複するニックネームがある場合>
システムは「同じニックネームのユーザーが存在しました。ニックネームを変更して下さい」メッセージを返す。

<プログラミング開始年月は適正な数値ではない場合>
システムは、「不正な値が記入されています。正しい数値を選択して下さい」メッセージを返す

<画像の大きさが適切でない場合>  
システムは「画像の大きさが適切ではありません。画像を選択し直して下さい」メッセージを返す。
```



`自身の「顧客プロフィール詳細」ページの表示`
```
【基本コース】
顧客は、「ナビゲーションバー」から「アイコンボタン」をクリックする。
システムは顧客のポートフォリオが存在すれば、それらを顧客プロフィールと共に取得し「顧客プロフィール詳細ページ」を表示する。

【代替コース】
<顧客のポートフォリオが存在が存在しない場合>
システムは、「まだポートフォリオは投稿されていません。」メッセージを返す。
```


`他の顧客の「顧客プロフィール詳細」ページの表示`
```
【基本コース】
システムは「ポートフォリオ詳細ページ」を表示する。
顧客は「ユーザーアイコン」をクリックする。
システムは、当該顧客のプロフィール情報を取得し、「顧客プロフィール詳細」を表示する。

【代替コース】
おそらくなし
```

`顧客へのお気に入り登録(削除)`
```
システムは、当該顧客のプロフィール情報を取得し、「顧客プロフィール詳細」ページを表示する。
顧客は、当該顧客の「プロフィールカード」内にある「お気に入り」ボタンをクリックする。
システムは、顧客がすでに「お気に入り」ボタンを押していたかを検証し、まだ「お気に入り」ボタンを押していない場合、お気に入り登録を行う。

【代替コース】
<顧客がすでに「お気に入り」ボタンを押していた場合>
システムは、お気に入り登録を抹消する。
```

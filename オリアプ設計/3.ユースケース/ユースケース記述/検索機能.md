【検索機能編】

`ユーザー検索`
```
【基本コース】
顧客は、「ナビゲーションバー」から「検索」ボタンをクリックする。
システムは「検索」ページを表示する。
顧客は、「検索タブ」から、ユーザー・ポートフォリオ・カテゴリ欄からユーザーを選択し、「検索フォーム」に検索キーワードを入力する。
システムは、検索キーワードに一致する顧客情報を一覧で「検索結果欄」に表示する。
顧客は、結果一覧からユーザーを選択する。
システムは、「顧客プロフィール詳細」ページを表示する。

【代替コース】
<「検索フォーム」に入力がない場合>
システムは「検索」ボタンを押させない。

<検索キーワードに一致する情報が存在しない場合>
システムは、「検索結果欄」に「「OO」に一致する結果はありませんでした。」と表示する。
```

【ポートフォリオの詳細ページ編】
`ポートフォリオの評価を投稿`
```
【基本コース】
顧客は、現在表示されている「ポートフォリオ詳細」ページ内で、ポートフォリオ評価をスキル・ビジネス性・創造性・社会性・利便性の各項目それぞれで、1.0~5.0の間で指定し、「送信」ボタンをクリックする。
システムは、評価が1.0~5.0の間にあるか、顧客がすでに評価を投稿していないか確認する。
そして、評価をデーターベースに保存する。
その後、システムは、新たな評価をもとにポートフォリオの評価の平均値を求める。
そして、システムは評価の平均値をポートフォリオ一覧画面で出力する。

【代替コース】
<評価が1.0~5.0の間にない場合>
システムは「評価は1.0~5.0の間で選択して下さい」メッセージを表示する

<スキル・ビジネス性・創造性・社会性・利便性の各項目のいずれかで顧客の入力がない場合>
システムは、顧客に「送信」ボタンを押させないようにする。

<顧客がすでに一度評価を投稿している場合>
システムは、当該ポートフォリオの当該顧客の評価を更新する。
```

`ポートフォリオへのお気に入り登録(削除)`
```
【基本コース】
顧客は「ポートフォリオ詳細」ページ内にある「お気に入り」クリックする。
システムは、顧客がすでに「お気に入り」ボタンを押していたかを検証し、まだ「お気に入り」ボタンを押していない場合、お気に入り登録を行う。

【代替コース】
<顧客がすでに「お気に入り」ボタンを押していた場合>
システムは、お気に入り登録を抹消する。
```

`チャットでのメッセージ投稿`
```
【基本コース】
システムは、「ポートフォリオ詳細」ページを表示する。
ユーザーは、「チャット欄」から1文字以上のメッセージを入力し「送信」ボタンをクリックする。
システムは、1文字以上のメッセージかメッセージの長さが長すぎないか検証する。その後、データベースにメッセージを保存し、投稿されたメッセージを「チャット欄」に反映させる。

【代替コース】
<メッセージが長すぎた場合>
システムは「送信」ボタンを押させない。

<メッセージが0文字の場合>
システムは「送信」ボタンを押させない。
```

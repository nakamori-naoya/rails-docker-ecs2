`顧客新規登録`
```
【基本コース】
システムは、「ログイン・新規登録」ページを表示する。
顧客は、顧客情報を入力し、「新規登録」ボタンをクリックする。
システムは、メールアドレスのフォーマットやメールアドレスが一意なものか検証し、パスワードと確認用パスワードが一致するか検証し、「顧客プロフィール作成」ページを表示する。

【代替コース】
<メールアドレスのフォーマットが正しくない場合>
システムは、「送信」ボタンを押せないようにする。

<メールアドレスがすでに登録されている場合>
システムは「メールアドレスはすでに登録されています。」メッセージを返す。

<パスワードと確認用パスワードが一致しない場合>
システムは、「送信」ボタンを押せないようにし、「パスワードが一致しません」メッセージを返す。
```



`顧客ログイン`
```
【基本コース】
システムは、「ログイン・新規登録」ページを表示する。
顧客は、顧客情報を入力し、「ログイン」ボタンをクリックする。
システムは、メールアドレスとパスワードに一致する顧客情報が存在するか検証し、「ポートフォリオ一覧」ページを表示する

【代替コース】
<メールアドレスとパスワードに一致する顧客情報が存在しない場合>
システムは、「メールアドレスとパスワードが正しいか確認して下さい」メッセージを表示する。

<顧客がパスワードを忘れた場合>
顧客は、「私のパスワードは？」ボタンをクリックする。
システムは、メールアドレス入力画面を表示し、顧客は、メールアドレスを入力し「送信」ボタンをクリックする。
システムは、アカウント情報を取得し、顧客のメールアドレス宛にパスワードを顧客にメールする。
```



`顧客ログアウト`
```
【基本コース】
顧客は、「ナビゲーションバー」から「ログアウト」ボタンをクリックする。
システムは、顧客のセッション情報を削除し、「ログイン・新規登録」ページを表示する。

【代替コース】
おそらくなし。
```
